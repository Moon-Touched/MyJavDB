MongoDB和爬虫的练手项目。
## 功能
1. 单个电影信息的抓取（番号，演员，标签，网址，磁链）
2. 单个演员信息的抓取（名字，作品数量，所有作品的网址）
3. 抓取一个演员的所有作品
4. 抓取收藏列表的所有演员信息
5. 搜索数据库中的磁链转存到PikPak网盘，并以番号命名存储文件夹
6. 将本地文件与数据库匹配，生成nfo文件，抓取封面。由于下载的视频文件名不规范可能导致识别失败，选择根据第一层文件夹的名字识别。所以网盘转存以番号命名存储文件夹，这样直接无脑下载。
